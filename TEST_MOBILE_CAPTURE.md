## 🐛 일반적인 오류 및 해결

### 오류 1: "서버 연결에 실패했습니다"
**원인:** API 서버가 실행되지 않음
**해결:** `python api_server.py` 실행

### 오류 2: "카메라에 접근할 수 없습니다"
**원인:** 카메라 권한 거부
**해결:** 휴대폰 설정 > 브라우저 > 카메라 권한 허용

### 오류 3: "연결 끊김"
**원인:** 네트워크 문제
**해결:** 노트북과 휴대폰이 같은 WiFi에 연결되어 있는지 확인

### 오류 4: "조건이 충족되지 않음"
**원인:** 촬영 환경 미흡
**해결:**
- 휴대폰을 더 평평하게
- 조명 추가
- 카메라 높이/거리 조정

### 오류 5: "QR 코드 생성 실패"
**원인:** QRCode.js 라이브러리 로드 실패
**해결:** 인터넷 연결 확인 (CDN 사용)
